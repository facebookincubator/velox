January 2022 Update¶

Core Library¶

  • Add support for bucket-by-bucket query execution to allow for lower memory footprint for joins and aggregations on bucketed-by keys.

  • Add support for converting ROW vectors (structs) to and from Arrow arrays.

  • Add support for variadic arguments to simple functions.

  • Add support for LEFT merge join.

  • Add support for AS statement to PlanBuilder.

Hive Connector¶

  • Add support for AWS S3 IAM Roles.

Presto Functions¶

  • Add pi(), to_base(), array_join(), regexp_replace() functions.
