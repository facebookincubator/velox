April 2024 Update¶

Documentation¶

  • Document operations on decimals for Presto and Spark.

  • Document spill write stats. #9326

Core Library¶

  • Fix bugs in Window operator. #9476, #9271, #9257

Presto Functions¶

  • Add word_stem() and to_iso8601() scalar functions.

  • Add support for DECIMAL inputs to arbitrary(), min() and max() aggregate functions.

  • Fix json_extract() for paths with wildcards.

Spark Functions¶

  • Add array_size(), flatten(), year_of_week() scalar functions.

  • Add collect_list() and regr_replacement() aggregate functions.

Hive Connector¶

  • Add support for storing decimal as integer in Parquet writer.

  • Add hive.s3.connect-timeout, hive.s3.socket-timeout and hive.s3.max-connections configs. #9472

  • Fix complex type handing in Parquet reader. #9187

  • Fix DWRF reader to skip null map keys.

Performance and Correctness¶

  • Add aggregation and window fuzzer runs to every PR.

  • Add nightly run of window fuzzer.

  • Add check for aggregate function signature changes to every PR.

  • Add biased aggregation fuzzer run for newly added aggregate functions to every PR.

Build System¶

  • Add nightly job to track build metrics.
