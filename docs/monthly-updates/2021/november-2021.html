November 2021 Update¶

Documentation¶

  • Add How to add lambda function? guide.

Core Library¶

  • Add basic support for inner merge join for the case when join inputs are sorted on the join keys.

  • Add AssignUniqueId operator to enable TPC-H query 21.

  • Optimize simple functions for inputs of type ARRAY and MAP.

Presto Functions¶

  • Add infinity(), is_finite(), is_infinite(), nan(), is_nan() functions.

  • Add parse_datetime(), quarter(), year_of_week(), yow() functions.

  • Add array_duplicates(), slice(), zip() functions.

  • Add rpad(), lpad(), split_part() functions.

  • Add url_extract_host(), url_extract_fragment(), url_extract_path(), url_extract_parameter(), url_extract_protocol(), url_extract_port(), url_extract_query() functions.

  • Add approx_set(), empty_approx_set(), merge() and cardinality() functions.

  • Add support for DATE inputs to date extraction functions to enable TPC-H queries 7, 8, and 9.
