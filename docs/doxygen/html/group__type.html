Adds Velox Python Bindings to the module m.

This function adds the following bindings:

  • velox::TypeKind enum
  • velox::Type and its derived types
  • Basic functions on Type and its derived types.
Parameters
mModule to add bindings too.
asLocalModuleIf true then these bindings are only visible inside the module. Refer to https://pybind11.readthedocs.io/en/stable/advanced/classes.html#module-local-class-bindings for further details.
